# УРОК / Таймер занятия

Минималистичный веб-таймер по таймингам занятия. Работает офлайн, ставится на
домашний экран (PWA). Адаптив под вертикаль и горизонталь.

## Шаблоны

- **Уроки:** 60 мин, 90 мин (с блоком «Я знаю что делать»)
- **Психология:** группа 45, группа 90, индивидуальная консультация 50
  (структура: приветствие → разогрев → основная часть → рефлексия → ритуал прощания)

## Файлы

- `index.html` — само приложение
- `manifest.webmanifest`, `sw.js` — PWA (офлайн + установка на экран)
- `icon.svg`, `icon-192.png`, `icon-512.png`, `icon-180.png` — иконки

## Деплой на GitHub Pages

1. Создай репозиторий, залей все файлы в корень (не в подпапку).
1. Settings → Pages → Source: `Deploy from a branch`, ветка `main`, папка `/ (root)`.
1. Через минуту приложение будет по адресу
   `https://ИМЯ.github.io/НАЗВАНИЕ-РЕПО/`
1. На iPhone/iPad: открой в Safari → Поделиться → «На экран „Домой”».

> PWA (офлайн и установка) работает только по https — то есть на GitHub Pages,
> а не при открытии файла локально.
